# Роботизированная кисть с силомоментной обратной связью

## 1. Общее описание проекта

Проект представляет собой прототип трёхпалой роботизированной кисти с системой силомоментной обратной связи. Все три пальца активные и подвижные, каждый имеет две фаланги и независимый сервопривод. Кисть способна захватывать предметы различной формы — от цилиндрических до плоских — благодаря системе адаптивного схвата с контролем усилия. Управление осуществляется через Bluetooth (мобильное приложение) или по USB (режим отладки с ПК).

## 2. Механическая конструкция и материалы

**Корпус:** Ладонь и фаланги напечатаны на 3D-принтере из пластика PETG (повышенная прочность и термостойкость по сравнению с PLA).

**Сборка:** Все детали соединяются винтами M3 с потайной головкой и самоконтрящимися гайками — конструкция разборная и ремонтопригодная.

**Тяга:** Стальная рыболовная проволока (струна) диаметром 0,4 мм. Протянута через каналы внутри фаланг и закреплена на качалках сервоприводов.

**Возврат:** Пружины растяжения из нержавеющей стали, установленные на тыльной стороне каждого пальца. Обеспечивают мгновенный возврат пальцев в исходное положение при отключении питания.

**Шарниры:** Металлические оси диаметром 2 мм с латунными втулками — исключают люфт и трение пластика о пластик.

**Защита:** Силиконовые накладки на кончиках пальцев для увеличения трения при захвате.

## 3. Электронная архитектура и обратная связь

**Контроллер:** Arduino Mega 2560 — выбран из-за большего количества ШИМ-каналов и аппаратных таймеров для одновременного управления тремя сервоприводами.

**Датчик усилия (силомоментная обратная связь):** Тензорезистивные датчики давления (FSR-402), закреплённые на внутренней стороне кончиков пальцев. При захвате предмета сопротивление датчика падает пропорционально приложенной силе — контроллер считывает аналоговый сигнал и ограничивает усилие сервопривода, предотвращая повреждение хрупких объектов.

**Датчик касания:** Инфракрасный датчик препятствия (TCRT5000), встроенный в центр ладони. Срабатывает при приближении объекта на расстояние менее 2 см, запуская алгоритм автоматического захвата.

**Индикация:** RGB-светодиод на тыльной стороне ладони:

| Цвет | Состояние |
|------|-----------|
| Синий | Ожидание команды |
| Зелёный | Захват выполнен успешно |
| Красный | Превышение усилия / ошибка |

**Связь:** Модуль HC-05 Bluetooth для управления со смартфона и USB-UART мост (встроен в Arduino Mega) для отладки с компьютера.
